Attack Surface Reduction (ASR)

‚úÖ Available Now

How it works: Prevents exploitation by enforcing strict device configurations.

Capabilities: Blocks suspicious command-line executions and file activities

Key ASR Rules:

  • Block credential stealing from Windows local security authority
  • Block process creations originating from PSExec and WMI commands
  • Block untrusted and unsigned processes from USB
  • Block executable files from running unless they meet prevalence criteria
